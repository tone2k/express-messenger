Introduction

Express Messenger is a messaging application built using Node, Express, Socket.io, JWT Auth, and MongoDb.

Also a sprinkle of boostrap!

What does it do?

It allows a user to Create an Account, Log into that Account and post Anonymous Messages out into the world.

More About 'It'

Why does It do...it?

It does it because it is good to express yourself! Also, Node and Express are good too so it works!

How does it do it?

The user accounts are created using JWT authentication tokens and stored to a database using MongoDB. The messages are pushed to web sessions using Socket.io with jQuery and server is running using Node and Express!

What's next for it?

I'm thinking to refine the CSS and remove Bootstrap completely. I want to tighten up the state management using React as well.

...and actual features?
